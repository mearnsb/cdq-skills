## Code Files

### 1. scripts/bulk_onboard_loop.py
- **Purpose**: Automated manifest generation and orchestration
- **Type**: Python script
- **Inputs**:
  - Schema name (default: "samples")
  - Action type (list-tables, generate-manifest, check-progress)
  - Limit (max tables, default: 200)
- **Outputs**:
  - `onboarding-manifest.json` (tables + suggested rules)
  - Progress updates to console
  - `.onboarding-progress.json` (progress tracking)
- **Features**:
  - Analyzes 200+ tables
  - Generates 10 context-specific rules per table
  - Handles failures gracefully
  - Progress tracking
  - Status reporting
- **Usage**:
  ```bash
  python scripts/bulk_onboard_loop.py --schema samples --action generate-manifest
  python scripts/bulk_onboard_loop.py --schema samples --action check-progress
  ```

## Generated Artifacts

### 1. onboarding-manifest.json
- **Purpose**: Complete table list with suggested rules
- **Generated by**: `scripts/bulk_onboard_loop.py`
- **Structure**:
  ```json
  {
    "schema": "samples",
    "generated": "ISO timestamp",
    "tables": [
      {
        "name": "table_name",
        "schema": "samples",
        "logical_name": "ONBOARD_CDQ_AUTO_samples.table_name",
        "row_count": 1000,
        "column_count": 5,
        "columns": ["col1", "col2", ...],
        "suggested_rules": [
          {"name": "rule1", "description": "...", "sql": "..."},
          ...
        ]
      }
    ],
    "summary": {"total": 200, "previewed": 200, "failed": 0}
  }
  ```
- **Usage**: Reference for execution, read suggested rules, run Phase 1-3

### 2. .onboarding-progress.json
- **Purpose**: Track progress across sessions
- **Auto-created**: First run of script
- **Structure**:
  ```json
  {
    "start_time": "ISO timestamp",
    "last_updated": "ISO timestamp",
    "completed": ["table1", "table2", ...],
    "in_progress": "current_table",
    "failed": ["failed_table1", ...],
    "pending": ["table_n", ...],
    "summary": {"total": 200, "completed": 10, ...}
  }
  ```
- **Usage**: Monitor progress, resume interrupted sessions
